# Lorem Ipsum Generator

A simple web application that generates Lorem Ipsum text for your design and development needs.

## Features

- Generate 1-50 paragraphs of Lorem Ipsum text
- Copy all generated text to clipboard with a single click
- Responsive design that works on both desktop and mobile devices

## How to Use

1. Open `index.html` in your web browser
2. Enter the number of paragraphs you want to generate (default is 5)
3. Click the "Generate" button or press Enter
4. The Lorem Ipsum text will appear in the output area
5. To copy the text, click the "Copy to Clipboard" button

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- Lorem Ipsum npm package (via CDN)

## File Structure

- `index.html` - Main HTML file
- `style.css` - CSS styles
- `script.js` - JavaScript functionality
