# Weather App

A simple, responsive weather application built with HTML, CSS, and JavaScript that uses the WeatherAPI.com API to display current weather conditions and forecasts.

## Features

- Current weather information (temperature, condition, wind, humidity, precipitation)
- 8-hour forecast
- Search for any city worldwide
- Responsive design for all device sizes
- Error handling for invalid locations

## Setup

1. Sign up for a free account at [WeatherAPI.com](https://www.weatherapi.com/) to get your API key
2. Open the `script.js` file and replace `'YOUR_API_KEY'` with your actual API key
3. Open the `index.html` file in your browser to use the app

## API Integration

This app uses the following WeatherAPI.com endpoints:

- `/forecast.json`: Gets current weather and forecast data

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- [Font Awesome](https://fontawesome.com/) for icons
- [WeatherAPI.com](https://www.weatherapi.com/) for weather data
