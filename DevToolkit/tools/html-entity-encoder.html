About HTML Entities:

HTML entities are special codes used to display reserved characters and symbols in HTML. For example, the less-than sign (<) is represented as &lt; to prevent it from being interpreted as the start of an HTML tag.

When to use HTML entities:

  • Security: To prevent XSS (Cross-Site Scripting) attacks when displaying user-provided content.
  • Display code: When you need to show HTML syntax as text rather than have it rendered.
  • Special characters: To correctly display characters like copyright (©), trademark (™), or non-ASCII characters.

All processing is done in your browser - your data never leaves your device.

HTML Entity Encoding Examples

HTML Code Example

Original:
<div class="container">Hello</div>

Special Characters Example

Original:
5 > 3 && 2 < 4 is "true" in JavaScript

5 &gt; 3 &amp;&amp; 2 &lt; 4 is &quot;true&quot; in JavaScript
