About Hex Colors

Hexadecimal color codes are used in HTML, CSS, and other web design contexts to represent colors. The format consists of a hash symbol (#) followed by six hexadecimal digits.

Each pair of digits represents the intensity of red, green, and blue components (RGB) of the color, with values ranging from 00 to FF (0-255 in decimal).

#FF0000
Pure Red (RGB: 255, 0, 0)
#00FF00
Pure Green (RGB: 0, 255, 0)
#0000FF
Pure Blue (RGB: 0, 0, 255)

Hex codes can also be written in shorthand format using three digits (e.g., #F00 for red) when each color component has two identical digits.

Color Theory Tips

  • Complementary colors sit opposite each other on the color wheel and create strong contrast (e.g., blue and orange)
  • Analogous colors are next to each other on the color wheel and usually create harmonious designs (e.g., blue, blue-green, green)
  • Triadic colors are evenly spaced around the color wheel and tend to be vibrant (e.g., red, yellow, blue)
  • Monochromatic colors are variations in lightness and saturation of a single color
  • The 60-30-10 rule suggests using 60% of a dominant color, 30% of a secondary color, and 10% of an accent color
