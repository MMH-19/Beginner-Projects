About Border Radius:

The CSS border-radius property defines how rounded the corners of an element should be. You can set different values for each corner to create unique shapes.

Browser Support:

The border-radius property is supported in all modern browsers (Chrome, Firefox, Safari, Edge).

All processing is done in your browser - your data never leaves your device.
